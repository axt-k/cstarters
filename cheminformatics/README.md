# Cluster Lipopeptide Starter Structures

This script performs cheminformatics analysis and visualization of lipopeptide starter structures and saves a figure `cheminformatics_analysis.png` together with its SVG version `cheminformatics_analysis.svg`.

---

## One-shot setup of environment (Python 3.12)

1) Ensure you have [mamba](https://mamba.readthedocs.io/) or conda installed.  
2) Create and activate the environment:

```bash
# Faster:
mamba env create -f environment.yml
mamba activate cstarter

# Or with conda:
conda env create -f environment.yml
conda activate cstarter
```

## Running the script

Run the script using Python:

```bash
python cheminformatics_analysis.py -h
```
